# Plate Calculator

A simple in-browser plate calculator for gym settings.

- Enter a target weight and see exactly which plates to load.
- **Barbell mode**: splits the load symmetrically across the bar with an optional 45 lb bar.
- **Single-stack mode**: stacks plates on a single peg (for plate-loaded machines), no symmetry.
- Color-coded plate animation with the standard lb inventory: 45 / 25 / 10 / 5 / 2.5.

## Usage

Open `index.html` in any modern browser — or visit the live page once GitHub Pages is enabled.

No build step, no dependencies.
